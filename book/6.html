Let's talk strings. And these ones will be the least silly. In fact they behave exactly as one would inspect. In ruby, we have many ways to do things with strings. We can deepen our grokking of this later, for now, we must meditate upon the simple things.

Let us start with a single quoted ('there is a horse is aisle 5') strings. If we take the string we just had, paste it in the interpreter, and run .methods.count On it, we would get 164, however if we change it to double quotes, we could get 164 as well. So it appears that we can use the same functions on both. That's nice, no need to learn weird rules. There are many ways to make strings and text in Ruby. We have everything from literals like %q and %Q for representing single and double quoted strings respectively, to heredocs using
